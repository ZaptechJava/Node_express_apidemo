# Sample CRUD rest api with express 


## Using this project

Step 1

```bash
$ npm install
```

Step 2 

```bash
$ node index.js 
```

## CRUD URLS

Get all users

```bash
http://localhost:3000/api/users
```

Add user 

```bash
http://localhost:3000/api/adduser
```

post params
 
firstName: john
lastName:doe
age:27

DELETE USER

```bash
http://localhost:3000/api/deleteuser/3
```

ID is passing in url parameters


UPDATE USER(PATCH)

```bash
http://localhost:3000/api/updateuser/
```

post param

id:1
firstName:john
lastName:connor
age:26
